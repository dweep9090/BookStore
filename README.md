# AI Powered BookStore

A full-stack bookstore platform built with modern web technologies and AI-powered features.

## Tech Stack

### Frontend

* React
* TypeScript
* Vite
* Tailwind CSS

### Backend

* Node.js
* Express
* TypeScript
* Prisma ORM

### Database & Infrastructure

* PostgreSQL (Neon)
* Redis (Upstash)

### AI

* Google Gemini

## Features

### User Features

* Authentication & Authorization
* Browse Books
* Search & Filtering
* Wishlist
* Shopping Cart
* Order Management
* Reviews & Ratings
* AI Book Summaries
* AI Book Recommendations

### Admin Features

* Dashboard Analytics
* Book Management
* Order Management
* Stock Logs
* Inventory Tracking

### Security & Performance

* JWT Authentication
* Role-Based Access Control
* Helmet Security Headers
* Rate Limiting
* Redis Caching

## Future Enhancements

* Real Payment Gateway Integration
